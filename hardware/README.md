# Hardware Prototype

This folder contains the hardware prototype of the SmartKissan project. The prototype was built using a Raspberry Pi to automate irrigation and support crop monitoring.

## Components Used

- Raspberry Pi 4
- Raspberry Pi Camera
- 4-Channel Relay Module
- 5V DC Water Pump
- Power Supply
- Connecting Wires

## How It Works

1. The Raspberry Pi receives commands from the web dashboard.
2. The relay module switches the water pump on or off.
3. The Raspberry Pi Camera captures plant images for disease detection.
4. The irrigation system can be controlled in both manual and automatic modes.
